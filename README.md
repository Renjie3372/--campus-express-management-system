# 校园快递管理系统

这是一个面向软件工程课程实践的校园快递管理系统文档仓库，重点服务于本地演示、课程答辩和后续实现落地。

## 仓库内容

- `docs/01_项目概述与功能范围.md`：项目定位、角色设计、核心功能和范围边界。
- `docs/02_最小数据库设计.md`：支撑课程演示的最小表结构和字段建议。
- `docs/03_本地演示脚本.md`：答辩或课堂展示时可直接照着走的演示顺序。
- `docs/04_开发落地建议.md`：前后端目录、技术路线和开发阶段建议。
- `docs/README.md`：文档导航和使用建议。

## 精简说明

本仓库已对原始 AI 生成文档做了收敛处理，只保留和课程项目最相关、最容易落地的核心内容：

- 去掉重复表达较多的扩展说明
- 去掉通用教程型材料
- 合并内容高度重叠的设计与演示说明
- 保留最适合课程展示和后续编码的主线文档

## 项目核心范围

- 登录与角色识别
- 普通用户、快递员、管理员三类角色主页
- 收件入库与用户取件
- 发件申请、审核分配与揽件确认
- 异常件登记与处理
- 基础记录查看
